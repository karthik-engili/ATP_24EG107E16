# Week 5 - CSS Layouts, Tailwind CSS & React.js Introduction

This week progresses from CSS layout techniques to modern frontend frameworks, covering Flexbox, Tailwind CSS, and React.js fundamentals.

The `flex_handson` folder contains an `index.html` and `style.css` demonstrating CSS Flexbox layout techniques for creating responsive, flexible page layouts with proper alignment and distribution of elements.

The `tables` folder includes `table_Ass.html` and `table_Ass.css`, which practice building styled HTML tables with CSS — covering borders, spacing, alignment, and visual formatting of tabular data.

The `Tailwind Project` folder is a full project setup using Tailwind CSS as a utility-first CSS framework, with `package.json` dependencies, a `src` directory for source files, and an `assets` folder containing images of Hyderabad landmarks used in the project.

`React_Product_Ass_1` is a Vite-powered React application that implements a product listing interface, demonstrating component-based architecture, props, and JSX rendering for displaying product data.

`React_User_Ass_2` is another Vite + React project focused on user data management, practicing state management, component composition, and dynamic rendering of user information.

The `assets` folder stores shared image resources (`.jpg`, `.webp`, `.avif`) used across the week's projects.
