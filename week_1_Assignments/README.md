# Week 1 Assignments

## Overview

This folder contains JavaScript fundamentals exercises from the first week. The assignments focus on conditionals, loops, array operations, functions, and object-array manipulation.

## Folder Contents

### `18-02-26`

This folder includes basic JavaScript practice problems centered on numerical comparisons and array processing.

- `larger_among_two_numbers.js` - compares two numbers and prints the larger one.
- `larger_among_three_numbers.js` - finds the largest of three numbers using conditional checks.
- `largest_among_3_numbers_function.js` - defines a reusable function to return the largest of three values.
- `sum_of_numbers_in_array.js` - calculates the total sum of numbers in an array using a loop.
- `smallest_number_in_array.js` - identifies the smallest number inside an array.
- `array_sum_by_function.js` - uses a function to compute and return the sum of an array.
- `searching_element_in_array.js` - searches for an element in an array and returns its index or indicates it is not found.

### `19-02-26`

This folder contains an employee management exercise using an array of objects.

- `Employee.js` - demonstrates array manipulation with object records:
  - insert a new employee object at a specific position using `splice()`
  - remove an employee by matching the `name` with `findIndex()` and `splice()`
  - update a nested `marks` array value for a specific employee using `find()`

## Learning Objectives

- Use `if` / `else` statements for decision-making
- Work with arrays and perform loops for aggregation and search
- Encapsulate logic in functions
- Manage arrays of objects and update nested values
- Apply array methods like `splice()`, `findIndex()`, and `find()`

## Notes

All examples are written in plain JavaScript and are suitable for execution in Node.js or browser console environments. They are intended as introductory practice for foundational programming concepts.
