JavaScript Fundamentals

This week covers the foundational concepts of JavaScript programming through hands-on practice exercises.

The `18-02-26` folder contains seven JavaScript question files (`Qs1.js` to `Qs7.js`) that focus on core topics such as comparing numbers, conditional statements, loops, functions, and array searching. These exercises build a solid understanding of variables, control flow, and basic function definitions in JavaScript.

The `19-02-26` folder includes `Employee_.js`, which demonstrates working with arrays of objects. It covers CRUD operations on an employee dataset — inserting a new employee at a specific position using `splice()`, removing an employee by name using `findIndex()`, and updating nested array values with `find()`.

Together, these assignments establish a strong base in JavaScript syntax, data manipulation, and problem-solving techniques essential for full-stack development.
