# Week 2 - Advanced JavaScript & Backend Introduction

This week dives into advanced JavaScript concepts, modular programming, and introduces backend development with Node.js and Express.

The `25-02-2026` folder contains three projects: **ECS** (an E-Commerce System with modules for cart, product, discount, and payment), **JS_Assignments** (eight real-world scenario scripts covering bank transactions, e-commerce carts, employee payroll, movie streaming, student performance, and temperature analysis using higher-order array methods), and a **Task_Management_system** built with modular JavaScript using separate files for tasks and validation.

The `26-02-2026` folder includes a **Library Book Management System** implemented in `Book.js` using class-based OOP, and **cloning assignments** that explore deep copy vs shallow copy of objects along with spread operator and destructuring exercises.

The `28-02-2026` folder introduces **Asynchronous JavaScript** (promises, async/await, OTP countdown and exam portal simulators), **Rest Parameters** for flexible function arguments, and a **Backend-1** project — a basic Express.js REST API server with route handling and HTTP request testing via `req.http`.

Overall, this week bridges the gap between frontend JavaScript mastery and server-side development fundamentals.
