# Week 4 - HTML & CSS Fundamentals

This week covers the building blocks of web development — HTML structure, semantic elements, and CSS styling.

The `HTML` folder contains six HTML files (`assgn1.html` to `assign4.html`, `index.html`, `simple_html.html`) that practice core HTML concepts including headings, paragraphs, forms, links, images, and page structure using semantic tags.

`Cat.html` is a standalone webpage about "Pochi The Cat" that demonstrates embedding images (base64 encoded), ordered and unordered lists, hyperlinks, and horizontal rules — all key HTML elements in a single page.

`nested_list.html` showcases nested unordered lists to display a hierarchical course listing (DSA, Web Technologies, etc.), demonstrating how to structure multi-level content in HTML.

The `bistro_html_css` folder contains a styled webpage (`index.html` + `styles.css`) — a restaurant/bistro themed page that applies CSS styling techniques including layout, typography, and visual design to an HTML structure.

`main_style.css` provides additional global CSS styling used across assignments, and `nested_list.html` practices nesting HTML list elements for organized content display.
